# Vehicle Owner QR Generator

A simple web-based tool to generate QR codes for vehicle owners. This can be used to display owner contact details on a vehicle's dashboard or window for identification or emergency purposes.

## Features
- **Easy Input:** Fields for Name, Vehicle Number, Mobile Number, and Emergency Contact.
- **Instant Generation:** Generates a high-quality QR code instantly.
- **Custom Header:** Displays "Owner's Details" above the generated QR code.
- **Downloadable:** Save the generated QR code as a PNG image.
- **Mobile Friendly:** Works on all devices with a modern web browser.

## How to Use
1. Open `index.html` in any web browser.
2. Fill in the Owner's Name, Vehicle Number, Mobile Number, and Emergency Contact Number.
3. Click **Generate QR Code**.
4. Use the **Download QR Image** button to save the code for printing or digital use.

## Technologies Used
- HTML5
- CSS3 (Vanilla)
- JavaScript
- [QRCode.js](https://davidshimjs.github.io/qrcodejs/) for QR code generation.
